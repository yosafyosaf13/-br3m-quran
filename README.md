# براعم تحفيظ القرآن الكريم - نسخة Next.js

## نظام إدارة حلقات تحفيظ القرآن الكريم
**مجمع التوحيد بالعباسة**

### المتطلبات
- Node.js 18+ أو Bun
- npm أو bun أو yarn

### التثبيت والتشغيل

```bash
# تثبيت المكتبات
npm install
# أو
bun install

# إعداد قاعدة البيانات
npx prisma db push
# أو
bun run db:push

# تشغيل الخادم المحلي
npm run dev
# أو
bun run dev
```

### بيانات الدخول الافتراضية (بيئة التطوير فقط)
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

### المميزات
- 📊 لوحة تحكم شاملة
- 👥 إدارة الطلاب والمعلمين
- 📋 متابعة الحضور والغياب
- 📖 تسجيل الحفظ والمراجعة
- 💰 إدارة المدفوعات
- 🏫 إدارة الحلقات
- 👁️ بوابة أولياء الأمور
- 📈 التقارير والإحصائيات
- 🔔 نظام الإشعارات
- ⚙️ الإعدادات والنسخ الاحتياطي
- 🔐 نظام المصادقة

### التقنيات المستخدمة
- Next.js 16 (App Router)
- TypeScript 5
- Tailwind CSS 4
- shadcn/ui
- Prisma ORM (SQLite)
- Recharts
- Zustand
- React Hook Form + Zod
